## Understanding Tokens

Tokens are the basic units processed by a Large Language Model. A token can represent:

- A word
- Part of a word
- A punctuation mark
- A special character

The model processes tokens rather than complete words.

### Example

```text
"Hi" → 1 token

"Hello, how are you?" → Multiple tokens
```

## Types of Tokens

### Prompt Tokens

These are the tokens contained in the input sent to the model.

**Example:**

```text
Who is Ajit Pawar?
```

### Completion Tokens

These are the tokens generated by the model in its response.

### Total Tokens

The total number of tokens used in a request.

```text
Total Tokens = Prompt Tokens + Completion Tokens
```

- Large Language Models process tokens rather than words.

- Token usage influences API cost and model limitations.

- The `response.usage` object provides detailed token statistics for every request.
